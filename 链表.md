链表

节点、指向下一个节点的next指针、头结点

节点：

~~~java
/**
 * 节点
 * @author jokerpz
 */
class Node {
	/**节点值*/
	private int data;
	/**指向下一个节点的指针*/
	private Node next;
	
	public Node(int data) {
		this.data = data;
	}
	
	public int getData() {
		return data;
	}
	public void setData(int data) {
		this.data = data;
	}


	public Node getNext() {
		return next;
	}
	public void setNext(Node next) {
		this.next = next;
	}
}

~~~
